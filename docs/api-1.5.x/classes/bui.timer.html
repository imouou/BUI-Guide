(
  • option
)
Object

Defined in /Users/wilson/personal/htdocs/opensource/bui-rollup/src/scripts/method/bui.timer.js:8

Parameters:

  • option Object
    • target String

      [时间显示在某个id 上]

    • onEnd Function

      [时间到以后执行回调]

    • [onProcess] Function optional

      [过程处理函数,参数为当前秒数]

    • [times] Number optional

      [多少秒, 默认:10]

    • [interval] Number optional

      [间隔,默认: 1000]

    • [reduce] Boolean optional

      [默认:true 递减 | false 递增]

Returns:

Object:

[ start 开始计时 | stop 停止计时 | pause 暂停 | restart 重新计时 ]

Example:

    var timer = bui.timer({
       onEnd: function (argument) {
           bui.alert("时间到")
       },
       times: 10
   })

   timer.start();
