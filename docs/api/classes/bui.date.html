                                                       // 03/20

toWeek

(
  • datestr
  • n
)

Defined in src/scripts/method/bui.date.js:293

某个日期为星期几, 或者周几

Parameters:

  • datestr String

    [ 日期 例如: 2020-03-20 ]

  • n Number | Array

    [ 默认不填返回 "星期几", 1为返回 "周几" ]

Example:

 var week = bui.date.toWeek("2020-3-20");
                                                       // 星期六
                                                       var week = bui.date.toWeek("2020-3-20",1);
                                                       // 周六
