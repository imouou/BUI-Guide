chainable

Inherited from bui.dialog: src/scripts/ui/bui.dialog.js:685

切换显示, 默认点一次打开,点2次关闭

Parameters:

  • [onOpen] [ 打开的回调, 非必需] Function optional
  • [onClose] [ 关闭的回调, 非必需] Function optional

Example:

       $("#id").click(function(){
                                                       // 1. 只做切换
                                                       uiDialog.toggle();
                                            
                                                       // 2. 切换带回调
                                                       uiDialog.toggle(function(){
                                                           // 打开的回调
                                                       },function(){
                                                           // 关闭的回调
                                                       });
                                                   })
