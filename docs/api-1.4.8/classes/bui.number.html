bui.number Class

Defined in: /Users/wilson/personal/htdocs/opensource/BUI 1.4.7/src/lib/ui/bui.number.js:7
Module: UI

数字增减条

支持静态初始化,常用于购物车

预览地址: demo

方法说明:

prev: 执行减少操作
next: 执行增加操作
value: 不传参则获取,传参则设置
disabled: 禁止输入修改
option: 获取设置参数
widget: 获取依赖控件

  • option

Defined in /Users/wilson/personal/htdocs/opensource/BUI 1.4.7/src/lib/ui/bui.number.js:7

Parameters:

  • option Object
    • [id] String optional

      [控件id, 不传则全部初始化,可以通过查找input的值设置.]

    • [min] Number optional

      [最小值]

    • [max] Number optional

      [最大值, 默认:100 ]

    • [step] Number optional

      [一次增加多少]

    • [value] String optional

      [初始值]

    • [disabled] Boolean optional

      [ 是否可以输入修改 false | true ]

    • [render] Boolean optional

      [ 是否动态填充, true | false ]

    • [autocheck] Boolean optional

      [ 1.4.8新增 自动校验 默认 true|false ]

    • [onInput] Function optional

      [ 1.4.3新增 输入数值的时候的实时校验处理 ]

    • [onChange] Function optional

      [ 1.4.8新增 改变的时候才处理 ]

    • [callback] Function optional

      [ 点击按钮的回调 ]

Example:

html:

       <div id="number" class="bui-number"></div>

js: 

       // 初始化
       var uiNumber = bui.number({
                       id:'#number'
           });
           //获取值
           uiNumber.value();
